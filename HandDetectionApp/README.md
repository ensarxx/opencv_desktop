# El Algılama Uygulaması - OpenCV

Bu uygulama, OpenCvSharp kütüphanesi kullanarak gerçek zamanlı el algılama yapan bir Windows Forms uygulamasıdır.


## Kullanılan Teknolojiler

- .NET 8.0
- Windows Forms
- OpenCvSharp4
- OpenCvSharp4.runtime.win
- OpenCvSharp4.Extensions

## Gereksinimler

- Windows 10/11
- .NET 8.0 SDK
- Webcam (USB veya yerleşik)
